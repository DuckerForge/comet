# Comet ☄️

Official website for **Comet** — a non-custodial multichain swap & bridge app built for the **Solana Seeker**, powered by the RocketX aggregator (200+ chains, 450+ exchanges).

Static landing page, no build step. Hosted on GitHub Pages.

- `index.html` — landing page
- `privacy.html` — privacy policy
- `styles.css` — styles
- `script.js` — minimal scroll/reveal enhancements
- `assets/logo.svg`, `favicon.svg` — brand assets

_Not affiliated with the Solana Foundation or RocketX._
